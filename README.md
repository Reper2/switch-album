# C.☆Penguin's Switch/2 Photo Album

Welcome to my personal collection of Nintendo Switch and Nintendo Switch 2 gameplay screenshot and video albums! 

Everything you need to browse, preview, and download the albums can be found on the main deployment:

👉 **[Visit the Website](https://reper2.github.io/switch-album/)**

---

## 📂 Project Structure

* **`/app`**: Contains the core application scripts and logic.
* **`/assets`**: Style sheets (`main.css`, `eggs.css`, alternate themes).
* **`/images`**: Metadata, favicon assets, and site graphics.
* **`injectLinks.js`**: Dynamic script for mapping and serving album download branches directly from GitHub.
